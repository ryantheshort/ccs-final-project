Feature Card

Admin
    -Admin can remove, edit, add profile info to all users. More features to come!

Account
    -User will be able to create an account with a custom username.

Start Scorecard
    -User can create a fully customizable scorecard. User can set the par, distance, and
    personal score per hole for them and up to three additional users.

Entire Live Scorecard
    -Even during their round, the user can view all of their holes at once, or view
    each hole by itself by simply adjusting their current page on the carousel menu.

Nearby Courses
    -Using Google Maps API, shows nearby courses within a certain local distance.

History
    -User will be able to see all previous scorecards played.
